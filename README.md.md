\#Student Performance Early Warning System



\## Project predicts the academic risk level of students using Machine Learning



\##Risk Levels

* Low Risk
* Medium Risk
* High Risk



\## Technologies Used

* Python
* Pandas
* Scikit- learn
* Flask
* Matplotlib
* Seaborn

\## Model Used



* Random Forest Classifier



\#model acuuracy 

95%



\##Features

* predicts student academic risk
* provides early warning messages
* simple web interface using Flask
